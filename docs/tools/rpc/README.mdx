# JSON RPC endpoints

An RPC node is a piece of infrastructure running blockchain client software, a server running both an Execution Layer (EL) and Consensus Layer (CL) infrastructure. 

## HTTP RPC endpoint(s)

### Gnosis

```jsx title="Gnosis Mainnet"
https://rpc.gnosischain.com/
```

```jsx title="Chiado Testnet"
https://rpc.chiadochain.net
```
:::info Testnet
Chiado is the official testnet, check the [specifications and information](/about/networks/chiado).
:::
